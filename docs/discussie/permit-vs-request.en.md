# 7.3. Health data permits vs. requests


The EHDS distinguishes between:

- Health data permits [art. 68](https://eur-lex.europa.eu/legal-content/NL/TXT/HTML/?uri=OJ:L_202500327#art_68)
- Health data requests [art. 69](https://eur-lex.europa.eu/legal-content/NL/TXT/HTML/?uri=OJ:L_202500327#art_69)
    
- Expectation is that a data request will be lighter and cheaper.
- Role of trusted data holders [art. 72](https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=OJ:L_202500327#art_72), who can execute calculations on the SPE and provide an answer to the data user
    - A trusted data holder managing a network of data stations can directly answer these questions
    - DHD and the Care Institute are _de facto_ such trusted data holders, as are IKNL, Vektis, RIVM and CBS.
