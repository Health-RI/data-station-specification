# 6.1.4. KIK-V implementation: infrastructure

## 6.1.4.1. The question-and-answer process via Nuts

Nuts provides the foundational infrastructure on which the KIK-V network is built. Its documentation consists of:

- a generic Nuts section, making use of the Nuts standards and reference implementation Nuts-node (maintained by Nuts, see the [Nuts documentation](https://nuts.nl/documentatie/))
- and a vendor specification specific to KIK-V, the so-called Nuts bolt. This is a technical elaboration of the KIK-V question-and-answer process via Nuts (Nuts application for KIK-V, maintained by KIK-V, see [documentation on the publication platform](https://kik-v-publicatieplatform.nl/documentatie/nuts-technischespecificaties/))
