## Processing Hub

At the time of writing the first version of this specification, substantial differences were identified in how, for example, the processing hub of KIK-V has been implemented compared to that of PLUGIN/vantage6. This is also one of the core questions of the ongoing [Health-AI](https://www.clinicaldatascience.nl/health-ai) breakthrough project.

For now, we therefore refer to the section on these two [implementations](../implementaties/index.md) for how this has been worked out in practice to date. We aim to describe more insights and lessons at a later stage and thereby arrive at further standardisation of the processing hub.
